Play around with variations on the Chaos game!  source  3D version
(hint: More vertices or less opacity requires more points. Less opacity exposes subtler structure.)

Define how the next point is calculated using a function of these variables:

  • currentX, currentY: Current point coordinates
  • targetX, targetY: Coordinates of randomly chosen vertex

The default function simply moves to halfway to the target. But you can use any of the arithmetic, trigonometry, or matrix functions in the docs of Math.JS, which is what evaluates the function.


Spirals! f(currentX, currentY, targetX, targetY) = [ dot([cos(pi/4), -sin(pi/4)], [(currentX + targetX)/2, (currentY + targetY)/2]), dot([sin(pi/4), cos(pi/4)], [(currentX + targetX)/2, (currentY + targetY)/2]) ]

Or just change 1/2 to something else.
